## 維護與更新

### 更新流程
1. 編輯筆記內容，確保格式與結構清晰，並遵循專案的 Markdown 標準
2. 在專案根目錄執行 `& "scripts\generate-commits.ps1"`
3. 可看到筆記最後更新的日期與時間，確保資訊的時效性
